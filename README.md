BattleC

BattleC é basicamente uma batalha de RPG de mesa dentro do seu computador. Cada classe tem suas vantagens (O Cavaleiro tem um escudo, se defende melhor; O Mago é mais poderoso, causa mais dano; O Arqueiro é mais ágil, se esquiva melhor; E o Bárbaro é impiedoso, tem mais chances de acertar os golpes.), você e seu oponente selecionam uma e iniciam o combate. É um combate simples, você pode Atacar(Rola 1d20 para ataque), Defender (Mitiga dano) ou Esquivar (Dificulta o inimigo de te atingir), vence quem derrotar seu oponente primeiro. 

- Para uma melhor experiência de jogo, altere a configuração de inicialização do Terminal (app) para "Host de Console do Windows". Para jogar basta executar o Server.exe, e em seguida executar o Cliente.exe duas vezes (duas janelas do terminal).
